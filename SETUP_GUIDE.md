Check if dataset loaded in backend (check startup logs)\n# Should show: ✅ Engine ready in XXs  (46000 movies, 40 clusters)\n\n# 2. If you see CORS errors:\n\n1. Update `main.py` line ~240:\n```python\napp.add_middleware(\n    CORSMiddleware,\n    allow_origins=[\"http://localhost:3000\"],  # Add specific origin\n    allow_methods=[\"*\"],\n    allow_headers=[\"*\"],\n)\n```\n\n2. Restart backend\n\n### Issue: \"Slow performance / Long load times\"\n\n**Solution:**\n- First backend startup takes 30-60 seconds (loading SBERT embeddings)\n- Subsequent requests are faster\n- If consistently slow, check:\n  - Network latency: `ping localhost:8000`\n  - CPU/Memory usage during recommendation computation\n  - Try reducing `top_n` parameter\n\n---\n\n## Project Structure\n\n```\nMovie-Recommender-System/\n├── main.py                           # ✅ Updated with explainability endpoints\n├── movie_recommender_v3.py\n├── requirements.txt\n├── dataset/\n│   └── tmdb_movies_cleaned.csv\n├── .env                              # ⚠️ Create this with TMDB_API_KEY\n├── .gitignore                        # ⚠️ Add .env to this\n├── frontend/                         # 🆕 Entire new frontend\n│   ├── package.json\n│   ├── tsconfig.json\n│   ├── tailwind.config.js\n│   ├── next.config.js\n│   ├── .env.local                    # ✅ Already configured\n│   ├── .gitignore\n│   ├── README.md\n│   ├── src/\n│   │   ├── app/\n│   │   │   ├── layout.tsx\n│   │   │   └── page.tsx              # Main page\n│   │   ├── components/               # 8 UI components\n│   │   │   ├── Navbar.tsx\n│   │   │   ├── SearchBar.tsx\n│   │   │   ├── MovieInputPanel.tsx\n│   │   │   ├── ClusterInfoCard.tsx\n│   │   │   ├── MovieCard.tsx\n│   │   │   ├── ExplainabilityPanel.tsx\n│   │   │   ├── SimilarityMeter.tsx\n│   │   │   └── RecommendationCarousel.tsx\n│   │   ├── hooks/                    # Custom React hooks\n│   │   │   ├── useMovieSearch.ts\n│   │   │   ├── useRecommendations.ts\n│   │   │   └── useClustersInfo.ts\n│   │   ├── lib/\n│   │   │   ├── types.ts              # TypeScript interfaces\n│   │   │   ├── api.ts                # API client\n│   │   │   ├── store.ts              # Zustand store\n│   │   │   └── utils.ts              # Helper functions\n│   │   └── styles/\n│   │       └── globals.css           # Tailwind styles\n│   └── node_modules/                 # Dependencies (after npm install)\n└── README.md\n```\n\n---\n\n## Running Both Backend & Frontend\n\n### Option 1: Two Terminal Windows\n\n**Terminal 1 -
